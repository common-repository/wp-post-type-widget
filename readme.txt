== Description ==

WP Post Type & Widget is the perfect plugin to create Custom Post Types and custom Taxonomies and custom widgets in a user friendly way, just like managing your regular posts and pages.

= Features =

* Create Custom Post Types
* Add post support by type as you want
* Create Custom Taxonomy by post type
* Create Custom widget

== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin.
2. Upload the 'wp-post-type-widget' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and active the plugin.

= Install Via Admin Area =

1. In the admin area go to Plugins > Add New and search "WP Post Type & Widget"
2. Click install and then activate.

= How to Use =

1. Configure setting from plugin settings (WP Post Type & Widget)
